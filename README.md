# GkernelCI_docker
GkernelCI docker environment

## Quick start

Clone GKernelCI_docker repository  
`git clone --recursive https://github.com/GKernelCI/Gdocker.git`  
Change the *docker-compose.yml* and start docker-compose   
`docker-compose up -d`
